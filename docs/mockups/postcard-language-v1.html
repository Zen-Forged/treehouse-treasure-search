Postcard / paper visual language — proof of concept

Three additive frames on Home only. Each adds one layer to the previous frame so you can see the cumulative effect and pick a stopping point.

What you're judging

  • Frame A — Paper texture only. Subtle SVG noise overlay on existing paperCream bg. Zero new chrome. Tests: does texture alone do enough work to read as "paper"?
  • Frame B — A + debossed double-hairline frame at viewport edges. Letterhead/stationery feel. Tests: does the frame add postcard language without postmarks supporting it?
  • Frame C — B + static Crestwood stamp graphic in masthead area, slight rotation, ~78% opacity. Tests: does the stamp work as identity, or feel forced without a system to support it across malls?
  • Out of scope per Q2(c): corner postmark cancellation lines.
  • Out of scope per Q3(b): a real per-mall stamp system. Crestwood stamp is one static asset to see on screen, not infrastructure.
  • Out of scope per Q1(a): any screen besides Home.
