Strengths

  • Closest to Spotify/Strava canonical share-worthy bar
  • Highest brand recall — viewers WILL remember Treehouse
  • Most distinctive visual signature (unmistakable on a feed)
  • Best future brand-recognition payoff (when brand established)

Trade-offs

  • Pre-beta brand recognition is THIN — chrome reads as "shouted at" not "earned"
  • Vendor content compressed; some vendors will dislike the framing
  • Reads as "Treehouse's content with vendor stamp" not "vendor's content amplified"
  • Highest implementation cost — full chrome system across 5 card variants
  • Hardest to walk back if vendors push back ("our voice gets lost")
Recommendation Frame ii balanced co-brand. The reference scan strongly suggests Frame iii canonical (Spotify Wrapped / Strava lead with heavy chrome) — but those references are post-recognition brands where the chrome IS the value. Treehouse Finds is pre-beta; the chrome doesn't yet carry brand equity to spend. Frame ii honors the share-worthy bar (leaf glyph, green palette, signature aesthetic visible) while giving vendor content the room it needs to feel like the vendor's voice amplified, not Treehouse's content with vendor stamp. As brand recognition grows post-beta (via volume of these shares accumulating), the chrome itself becomes more share-worthy and the path to Frame iii becomes natural. Frame iii is a "next-year decision" the project earns into.
