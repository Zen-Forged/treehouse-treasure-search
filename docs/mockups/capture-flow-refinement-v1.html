Open picks for you

  1. Q1 — Masthead size. Frame 1 A / B / C above.
    My read: B (your spec). A is too modest given the "premium feel" framing; C is heavier than your +25px target.
  2. Q2 — /post/tag ready-state copy. Keep current "Now the tag" / "Capture the title and price in one shot." OR rewrite to match the /post/preview tonal shape (e.g. "Add the tag" / "We'll read the title and price for you.")?
    My read: keep current. The italic subtitle does real work explaining what the tag photo is for — first-time vendors need it.
  3. Q3 — Auto-grow caption cap. No cap (page scrolls when caption is huge) OR cap at viewport (~60vh, then scroll inside the textarea)?
    My read: no cap. AI captions average 2–4 sentences; a 10-sentence one is a corner case where natural page scroll is fine.
  4. Q4 — Polaroid label position. On Frame 2, labels "Find" and "Tag" sit below each polaroid (italic Lora muted). Alternative: put labels above each polaroid (more like a section header), or as a top-of-photo eyebrow inside the polaroid.
    My read: below each polaroid. Reads as caption-on-photo; doesn't compete with the page title above.
