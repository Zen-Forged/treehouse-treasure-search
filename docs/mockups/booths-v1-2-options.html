Booths Page — Layout Options

All three options use v1.2 chrome (IM Fell, paperCream, v1 tokens). Choosing one replaces the current v0.2 Georgia + legacy colors layout.

What are we deciding?

A — Updated List: Same single-column card structure as today, but migrated to v1.2 tokens and IM Fell. Most conservative. Fastest to ship. Booth name gets bigger (16px IM Fell vs current 15px Georgia). Zero structural change — just a chrome swap.

B — 2-Column Grid: Two booth cards per row. More density — you can see 6 booths in one screen instead of 3. Works well when there are 10+ booths. Each card shows the hero square, booth number chip, name, and mall. Trade-off: less room for bio text.

C — Editorial Cards: Taller hero (140px vs 92px), larger name (19px IM Fell), full 2-line bio below. Fewer booths per scroll, but each feels premium — more like a shop page than a directory. Best if the vendor photo quality is high.
