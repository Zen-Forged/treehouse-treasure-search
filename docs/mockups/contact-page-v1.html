Contact page — R7 v1

v0 mechanism is mailto with a category subject prefix. Three visual treatments for the picker. Pick one or redirect.

Decisions already locked

  • Mechanism: mailto link, no backend, no Tally subscription.
  • Categories: Question / Feedback / Bug — each prefills the email subject (?subject=Question:%20, etc.).
  • Discoverability: link added to /login below the "First time? An account…" line. Footer-everywhere is post-Wave-1 polish.
  • Recipient: the project's admin email (NEXT_PUBLIC_ADMIN_EMAIL).

What this mockup is asking you to judge

  • Visual treatment of the 3-option picker — Frame A filled green buttons, Frame B soft cards with icons, Frame C minimalist text rows.
  • Which best matches the brand voice (warm, parchment, literary serif)?
