What V1 spans

The mockup spans ★ Favorite Mall placement + distance pill placement. Four frames; everything else is held constant (typography, palette, row pattern, accordion shape, GET DIRECTIONS, "X finds waiting" copy + chrome, decorative leaf).

  • α — ★ top-right corner bubble · distance pill inline next to mall name
  • β — ★ inline with mall name · distance pill inline next to mall name
  • γ — ★ twin-button with GET DIRECTIONS · distance pill inline next to mall name
  • δ — ★ top-right corner bubble (α structure) · distance pill relocated to eyebrow above GET DIRECTIONS — pairs distance + directions semantically; mall name breathes at full width ← picked + refined

Production-vs-mockup notes

  • Wordmark — the stacked Cormorant + leaf-above + horizontal-rule treatment in the masthead is placeholder rendering only. Production uses the existing PNG asset (sessions 87/88/104, 4-session-hardened). v2 keeps the PNG unchanged; wordmark v2 redesign is a separate decision worth its own session if/when it surfaces.
  • Decorative leaf top-right — pulled out of Saved v2 scope. Will be applied universally as global page-background chrome in a separate v2 Arc covering all surfaces, designed to sit behind content without impacting any elements on top.

Frame δ refinements (post-pick)

  • Address spans full mall-card width edge-to-edge — restructured `head-δ` as a 2-row CSS grid (name + ★ on row 1; address spans both columns on row 2). Long addresses no longer truncate against the ★ bubble. Applies only to Frame δ; α/β/γ retain the original meta-block layout as historical-rejected reference.
  • Find-row meta drops the booth meta line — accordion header already names the booth (BOOTH 214 small-caps + italic name), so the per-find "Booth 214" tertiary line was redundant. Find-row meta is now just name (Cormorant 17px) + price (Inter 14px). Applies only to Frame δ's find rows.

Held constant across all frames

  • Typography: Cormorant Garamond (mall names 28px SemiBold, find names 17px medium, booth names italic 17px) + Inter (metadata, prices, buttons, navigation)
  • v2 palette: --bg-main #F7F3E8 page bg, --card-surface #FFFCF5 mall cards, --card-surface-warm #FBF6EA distance pill + thumbs, --accent-green #285C3C CTA
  • Row pattern: ✓ Found circle (24px) + 64px photo thumb + serif name / sans price / sans booth meta + ♥ Save leaf bubble (36px). Each row is a single-line height with multi-line stack inside.
  • Per-booth accordion: Storefront icon + BOOTH N small-caps + italic Cormorant booth name + chevron up/down. Tap to expand/collapse find rows below.
  • "X finds waiting to be discovered": Centered FONT_SERIF italic 13px with PiLeaf glyph prefix + dashed-rule flankers. Sits below GET DIRECTIONS, above accordion sections.
  • Decorative chrome: Subtle olive-green leaf-branch SVG top-right corner of every page; consistent placement.
  • Bottom nav: 3-tab — Explore / Saved (with 9+ badge) / Map. Saved is active.
