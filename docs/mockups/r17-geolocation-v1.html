R17 — Geolocation Enrichment v1

/find/[id] cartographic card with distance + Navigate + View on Find Map.
Three structural alternatives — pick the shape; pixel polish in V2 with system-wide rollout.

Frozen decisions (from session-117 standup)

  • Surface scope: system-wide — /find/[id], /shelf/[slug], /flagged, /shelves. V1 illustrates on /find/[id]; V2 will lock how the chosen shape attaches to BoothLockupCard + BoothHero.
  • Augment vs replace: augment — distance pill + CTAs bolt onto the existing cartographic card, the card itself is unchanged.
  • Permission UX: silent prompt on first mount of any surface using the primitive (whichever the user lands on first).
  • Permission-denied state: hide pill + CTAs entirely (silent failure). Card reverts to current state. Each frame shows this at the bottom for confirmation.
  • Address line: mall name + city only (mockup convention preserved). Street address is revealed by the deep-link, not on-screen.
  • Roadmap: R17 — Geolocation-aware discovery — promotes 🟡 → 🟢 once V1+V2 lock.
Existing infrastructure: malls.latitude + malls.longitude already populated for all 29 active malls (reverse-geocoded via scripts/add-mall.ts). PWA standalone-mode geolocation works in iOS Safari. /map?mall=<slug> already accepts mall scope (R10, session 108) — "View on Find Map" routes there.

Questions for V1

  1. Q1 — Pill anchor + CTA shape: which frame reads right?
    (A) top-right pill + twin buttons "View on Find Map" + "Navigate" — matches your reference
    (B) eyebrow-row pill + primary "Navigate" + text-link "View on map"
    (C) inline pill in mall subtitle + single combo "Find this booth on the map →"
  2. Q2 — Distance format:
    (a) "2.7 mi" with tiny ↗ glyph (mockup convention)
    (b) "2.7 mi" plain, no glyph
    (c) "2.7 mi · 8 min" — distance + ETA pair (Apple Maps style; requires Mapbox Directions API call)
  3. Q3 — Should the pill be tappable?
    (i) Pill is decorative — display only. CTAs do the work.
    (ii) Pill itself opens the deep-link to native maps app — 3rd CTA hidden in the pill.
