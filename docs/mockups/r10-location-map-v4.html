Refining v3 in two places: the stamp is square again (52×52, vertically centered), and the cancellation is wavy SVG ink instead of stacked hairlines — looks like an actual postal cancellation, not a UI list of bars.

v4 mockup · 2026-05-04 · square stamp · ink cancellation · same 4 tab variants

What changed from v3

  • Stamp is square (52×52) — was 56×full-content-height in v3 (rectangular). Now vertically centered as a flex item, smaller footprint, more recognizable as a postage stamp.
  • Cancellation is wavy SVG ink — 5 sinusoidal paths, ink-color stroke at 1.6px, low opacity (0.42), rotated −6° for hand-stamped feel. Reads like real postal cancellation ink.
  • Cancellation partial overlap on stamp's left edge — extends ~14px into the stamp + ~70px to the left across the paper area, mimicking how real cancellation ink crosses both the stamp and the surrounding card surface.
  • Footprint smaller — total cancellation+stamp area is now ~110px wide (was ~140px in v3 with stretched stamp + cancel-marks gutter).
  • Same dashed-border treatment on stamp rim — kept from v3. Renders reliably, gives the "perforated edge" read without per-pixel hacks.

Why wavy ink reads as "real cancellation"

Real postal cancellations are wavy parallel ink lines (called "killer bars") that cross both the postage stamp and the surrounding paper. They're hand-applied at slight angles and bleed slightly. Stacked horizontal hairlines feel schematic because they're too geometric — wavy curves at low opacity feel like ink on paper.

v4 uses 5 SVG paths with quadratic Bézier curves (Q + T) at the same baseline spacing, rotated −6° as a group. Stroke color is inkPrimary with 0.42 opacity to keep the stamp glyph readable through any partial overlap.

Surface

  • Card bg — postcardBg #f4ecd8 (new token; warmer than paperWarm).
  • Border — 1px solid rgba(42,26,10,0.10), radius 12px, padding 16px.
  • Layout — flex row, gap: 14px, align-items: center, position: relative (anchors cancellation).

Text block

  • "from:" — Lora italic 18 / inkMuted / line-height 1.
  • Mall name — Lora 26 / 500 / inkPrimary / line-height 1.15. WebkitLineClamp: 2.
  • Address — Lucide MapPin 13 + system-sans 13 / inkMid. Single line, ellipsis.
  • Right padding 28px — reserves space for the cancellation ink so address text doesn't run under wavy lines.

Cancellation (SVG)

  • 5 quadratic Bézier paths via Q + T at baselines y=4, 11, 18, 25, 32.
  • Stroke inkPrimary 1.6px, linecap: round, no fill.
  • SVG positioned absolute, right: 54px (~14px overlap onto stamp's left edge), top: 50%, transform: translateY(-50%) rotate(-6deg).
  • Width 84, height 38, opacity 0.42, z-index: 2 (over stamp), pointer-events: none.

Stamp (square)

  • Outer — 52×52, paperWarm rim, 1.5px dashed greenBorder, 4px radius, 4px padding.
  • Inner — green fill, 2px radius, glyph centered.
  • Glyph — 26px, greenOn stroke, 2px stroke. Same Lucide shape as the active BottomNav tab.
  • Position — flex item, align-self: center.
