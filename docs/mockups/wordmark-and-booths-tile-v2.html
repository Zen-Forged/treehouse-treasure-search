Booths row pattern — pick one

1. C1 or C2?
  • C1 — with the 56×56 photo thumbnail. Visual identity preserved.
  • C2 — text only. Densest, cleanest, photos live on /shelf/[slug] only.
2. Lockup color — keep green or unify with new ink-black wordmark?
  • Green (default) — booth eyebrow + numeral stay v1.green. Find Map booth-section-headers also stay green; the system reads consistent.
  • Ink-black — flip to v1.inkPrimary. Quieter; would also flip Find Map's lockup to match.
3. Bio line on rows — show when present?
  • Yes (as drawn) — italic 11.5px under the vendor name when bio is set; collapses to single-line layout when not.
  • No — vendor name only, single line, predictable row height across all rows.
