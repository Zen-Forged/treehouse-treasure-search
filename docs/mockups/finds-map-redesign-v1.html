  • D7. Closer "circle + tagline" footer = retired with the spine. Page just ends, like Home.

What carries from session 69

Direct vocabulary inheritance — the card header, eyebrow, vendor type, and tile caption treatment all match Booths grid + Find Detail rail + Find Detail booth card.

  • Card surface — v1.inkWash bg + v1.inkHairline 1px border + 8px radius + 10/12 padding (matches session-69 unified card treatment).
  • Eyebrow — FONT_SYS 9.5px / 700 / 0.10em letter-spacing / uppercase / inkMuted (matches Booths VendorCard + Find Detail booth card).
  • Vendor name — FONT_IM_FELL 18px / inkPrimary / -0.005em letter-spacing (matches Find Detail booth card; Booths grid uses 18px also).
  • Mall subtitle (when shown) — FONT_SYS 11.5px / inkMuted, single line, format: {mallName} · {locationCity}, {state}.
  • Tap target — whole card link to /shelf/[vendorSlug] when slug exists; falls back to non-tappable card with same visual when slug is missing (parallel to Find Detail booth card behavior).
