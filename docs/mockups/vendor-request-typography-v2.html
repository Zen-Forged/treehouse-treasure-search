What this mockup is asking you to judge

  • Which serif replaces IM Fell project-wide — across titles, intros, labels, find-tile captions, post-it booth names, ack copy, helper text. Wherever the current code says FONT_IM_FELL, the chosen font lands.
  • Italic legibility — the find-tile caption sample is the hardest test. IM Fell italic at 14px on a 4:5 tile is the surface David named as a recurring readability sore.
  • Hierarchy across sizes — title 24px / header 26px / labels 15px / caption 14.5px / post-it name 18px. The font has to hold on all of these.

Sweep mechanics: Replace FONT_IM_FELL in lib/tokens.ts with the chosen family. All consumers update at once. Numerals (FONT_NUMERAL = Times New Roman) and wordmark (also TNR) stay as-is. For Lora / Newsreader: add font-load <link> to app/layout.tsx. For Georgia: web-safe, no font load needed (immediate paint, zero FOUT).
Decide
  1. Which font replaces IM Fell?
    Lora Modern editorial · best italic · most polished body feel
    Newsreader Editorial w/ optical sizes · most flexible hierarchy · slightly newer
    Georgia Web-safe · most legible at small sizes · slightly less custom-feeling
  2. Title size + weight — V2 set page title to 24px / weight 500 (was IM Fell 22px / weight 400). The new fonts are slightly less wispy than IM Fell so weight 400 reads heavy enough — happy to drop to 22px / 400 if 24px / 500 feels too display.
  3. Ack-card copy — kept italic in V2 (matched the IM Fell italic primitive). Easy flip to upright if italic feels off in the new font.
