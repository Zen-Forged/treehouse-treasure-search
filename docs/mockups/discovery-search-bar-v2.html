Why binoculars beats magnifying glass

  • Magnifying glass is the universal database-search glyph — Google, Spotlight, every form on the internet. It signals "I'm querying a system."
  • Binoculars is the scouting / spotting / scanning the landscape glyph. It signals "I'm looking for something in a real place."
  • Composes with the Treehouse thesis ("digital tool for real-world exploration") and the existing leaf brand mark — both are field-vocabulary, not desk-vocabulary. Compass for entry-points, binoculars for spotting, leaf for keeping.
  • Phosphor's BinocularsFill reads cleanly at 22px and stays legible down to 18px.

Implementation

  • import { BinocularsFill } from "@phosphor-icons/react"; already a project dep (per the codebase's existing Phosphor usage on FlagGlyph + the BookmarkBoothBubble glyph swap session 81).
  • Slot it inside the SearchBar primitive. weight="fill", size={22}, color="var(--v1-ink-mid)".
  • Filter glyph on the right side stays SlidersHorizontal (regular weight, inert in phase 1).

Where else binoculars could earn a spot

  • Empty-state CTA. "Search all of Kentucky" could become "Look across Kentucky →" with a small inline binoculars glyph. Only if the verb shift reads better.
  • R10 map view. When the location-map nav lands, binoculars could be the "explore from here" affordance — same icon, same mental model.
  • Don't extend further yet. Phase 1 = one icon, one place. Promote to a vocabulary on the second firing.

What's still open

  • Should the placeholder copy shift to match the binoculars vibe? Spot finds, booths, or styles would compose better than Search…. Light recommendation: yes — but defer to David since "Search" is the universal mental model and "Spot" might read clever-for-clever's-sake.
  • Empty-state copy is unchanged from v1 ("Nothing matching 'X' at this location yet" + "Search all of Kentucky →"). Re-evaluate together if the placeholder shifts.
