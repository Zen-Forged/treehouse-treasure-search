"Treehouse Finds" wordmark — Times New Roman, upright, ink-black

Item 5 from David's session 80 redirect bundle. The wordmark sits in the masthead on every screen — this is a brand-visible swap so it's worth a stillframe before code.

What's changing

  • Font — IM Fell English (decorative serif) → Times New Roman (the same TNR already used for booth numerals). Already in tokens as FONT_NUMERAL; no new webfont load.
  • Style — italic → upright.
  • Color — v1.green (#1e4d2b, leaf green) → v1.inkPrimary (#2a1a0a, ink-black, the same color used for body type).
  • Lift — drop the existing transform: translateY(-1px). That nudge was a fix for IM Fell italic ascenders sitting above their line-box center; TNR upright sits true and the lift would push it visually high against the side bubbles.
